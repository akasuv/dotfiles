# dotfiles
Dotfiles for my macOS setup powered by [rcm](https://thoughtbot.github.io/rcm/rcm.7.html).

## Clone dotfiles to home root

```bash
  cd ~ && git clone ... && mv dotfiles .dotfiles
```

## Synchronize your home directory

```bash
  rcup -v
```

## Add new rc files

```bash
  mkrc .zshrc ...
```
